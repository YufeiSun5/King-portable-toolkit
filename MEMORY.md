- 修复前端空按钮问题，补齐项目/文件夹/CSV 树节点右键菜单和变量复制入口。
- 修复 Wails dev 黑屏：清理残留端口、补稳定启动脚本，并处理工作区树 null 数组崩溃。
- 调整搜索与工作区联动：搜索占用顶部栏，左侧树支持选中、展开/收缩，右侧展示当前层级变量。
- 统一弹框体系：新增 AppDialog/dialogStore，替换前端原生 prompt/alert，并补齐列批量操作弹框。
- 优化左侧树交互与样式：点击项目/文件夹文字即可选中并展开/收起，右键菜单沿用统一应用弹框。
- 修复主编辑区被提示面板挤压的问题，将主区域调整为 Header/工具面板/表格/状态栏纵向布局。
- 新增保存入口，支持保存当前文件和保存全部；新增 Git 图谱式还原点面板。
- 工作区树顶部新增新建项目按钮；文件夹内新建 CSV 会创建功能节点、自动展开并选中，支持同一文件夹多个 CSV。
- 字段模式改为简洁模式/全量模式：简洁模式显示关键列，全量模式按字段元数据展示所有列。
- 导出规则调整为始终导出全量字段；选中文件夹或项目时按树目录层级拆分为多个 CSV 下载。
- 搜索改为输入即时过滤，并增加列条件筛选构造器，支持“列等于/列包含”条件芯片。
- 移除右侧“当前层级变量”竖向列表，仅保留当前选中变量详情快编。
- 新增脏状态：表格或工作区变动后保存按钮出现水波纹背景，保存后恢复平静。
- CSV 功能节点创建/重命名/删除接入后端 SQLite；浏览器开发态无 Wails runtime 时保存到 localStorage，刷新后保留新建文件夹和 CSV。
- 修复筛选后的批量操作范围：单元格“应用到整列”和列头批量操作只作用于当前表格可见行，隐藏行不会被误改。
- 筛选结果变化后自动校正选中行；工具栏复制变量优先复制当前可见行，避免操作隐藏的旧选中行。
- 修复 Wails dev 依赖 npm 启动失败：wails.json 改为直接用 node 执行本地 Vite/TypeScript 脚本，适配 vfox Node SDK 缺少 npm.cmd 的环境。
- 表格新增 iOS 风格圆点选择列；未勾选时批量范围默认当前可见全部，勾选后只使用手动选择行。
- 新增“复制到文件夹”入口和弹框，可将当前有效范围内的一个或多个变量复制到目标文件夹。
- 删除主布局右侧详情面板，变量编辑统一回到表格内完成。
- 还原点打开后收起变量表，只显示当前选中文件或目录下的记录；左侧为细竖向记录线，右侧上方显示当前数据、下方显示选中还原点数据。
- 还原点数据展示改为上下两张同步滚动的只读对比表，列宽按内容自适应，新增/删除/修改分别用绿色/红色/黄色背景标识。
- 批量替换改为双输入框：填写“要替换的内容”和“替换成”，例如 `_b` 替换为 `_c` 会把 `A_b` 处理为 `A_c`。
- 还原点的“保存当前文件”对比改为使用表格状态中的已保存快照，变量名等任意列的编辑都能在上下表中以黄色修改态显示。
- 变量表行数据接入本地持久化：保存时写入 `king-portable-toolkit.kio-rows.v1`，刷新后复制到文件夹的变量不会丢失。
- KIO 导出列补齐为 51 个完整字段，按真实 KIO 表头顺序导出，并补充常用默认值与中文枚举值转换。
- 工具包导航改为顶部图标菜单：只有当前选中工具显示文字；默认进入欢迎页，KIO 选中时才显示左侧项目树，Ping/端口为占位页。
- 表格“操作”列显示三点菜单和最近一次行操作图标按钮；三点左键菜单改为 fixed 定位并移除无意义的编辑/查看入口。
- KIO 导入按钮接入系统文件选择器，支持解析 CSV/TSV 为变量行；导入同名文件时在同目录自动追加序号生成新 CSV 节点。
- 修复珍珠山 GB18030 CSV 导入不显示：分隔符按逗号/制表符数量识别，避免表头缩进里的制表符误判为 TSV；导出 CSV 增加 UTF-8 BOM，避免常见表格软件打开中文乱码。
- 还原点面板移除演示假记录，改为读取真实保存快照；打开时只显示当前选中文件或文件夹命中的保存记录，未保存的字段清空会和最近保存版本做黄色差异对比。
